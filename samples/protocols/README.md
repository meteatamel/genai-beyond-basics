# Agent Protocols

![MCP, A2A, A2UI, AG-UI](images/mcp-a2a-a2ui-agui.png)

This directory contains samples of some of the main agent protocols.

| Protocol | What's it? | Samples |
|:---------|:----------|:---|
| [Model Context Protocol](https://modelcontextprotocol.io/) | An open protocol, from Anthropic, that standardizes how models and agents access external tools and resources | [mcp](./mcp) |
| [Agent2Agent Protocol (A2A) ](https://a2a-protocol.org/latest/) | An open protocol, from Google, that standardizes how agents running on diverse frameworks and platforms communicate | [a2a](./a2a) |
| [Agent-User Interaction Protocol (AG-UI)](https://github.com/ag-ui-protocol/ag-ui) | An open protocol, from CopilotKit, that standardizes how agents communicate with user interfaces | [ag-ui](./ag-ui) |
| [Agent to UI Protocol (A2UI)](https://a2ui.org/) | A generative UI protocol, from Google, that enables AI agents to generate rich, interactive user interfaces across web, mobile, desktop. | [a2ui](./a2ui) |
